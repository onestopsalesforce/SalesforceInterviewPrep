JavaScript Function Parameters: By Value or Reference

In JavaScript, function parameters are passed by value. However, the behavior differs based on the type of the data being passed:

1. Primitive Types

Primitive types (e.g., number, string, boolean, undefined, null, bigint, symbol) are always passed by value. This means a copy of the value is passed to the function, and changes made to the parameter inside the function do not affect the original variable.

Example:

function modifyPrimitive(val) {
    val = 42; // Changing the local copy
    console.log("Inside function:", val); // 42
}

let num = 10;
modifyPrimitive(num);
console.log("Outside function:", num); // 10 (unchanged)

2. Objects and Arrays (Reference Types)

Objects and arrays are also passed by value, but the value being passed is a reference to the object in memory. This means changes to the object's properties inside the function do affect the original object.

Example 1: Modifying an Object Property

function modifyObject(obj) {
    obj.name = "Alice"; // Modifying the object
    console.log("Inside function:", obj);
}

let person = { name: "Bob" };
modifyObject(person);
console.log("Outside function:", person); // { name: "Alice" } (modified)

Example 2: Reassigning an Object

function reassignObject(obj) {
    obj = { name: "Charlie" }; // Reassigning to a new object
    console.log("Inside function:", obj);
}

let person = { name: "Bob" };
reassignObject(person);
console.log("Outside function:", person); // { name: "Bob" } (unchanged)

Key Takeaways:

  • Primitive Types: Always passed by value. Modifications inside the function do not affect the original variable.
  • Objects and Arrays: The reference to the object is passed by value. Modifying the object inside the function affects the original object, but reassigning the parameter does not affect the original object.

Simplified Rule

Primitive values: Immutable and passed as a copy.
Objects/arrays: References are passed by value, allowing modification of the original data but not reassignment.
